# Resume Version Selector — Quick Reference

## Three Versions Available

| Version | File | Best For |
|---------|------|----------|
| **V1 — VP/SVP Enterprise** | `resume_v1_vp_svp_enterprise.md` | Pharma, oncology, biotech VP/SVP roles with enterprise media budgets |
| **V2 — Healthtech/SaaS Pivot** | `resume_v2_healthtech_saas_pivot.md` | Healthcare technology companies, SaaS platforms, commercial tech roles |
| **V3 — Ultra-Executive** | `resume_v3_ultra_executive.md` | C-suite adjacent roles, recruiter submissions, board-level audiences |

---

## Decision Tree

```
Is the company a healthtech/SaaS platform?
├── YES → V2 Healthtech/SaaS Pivot
└── NO
    ├── Is the role SVP or C-suite adjacent at Fortune 500 / PE-backed?
    │   ├── YES → V3 Ultra-Executive
    │   └── NO
    │       ├── Is it a recruiter/executive search submission?
    │       │   ├── YES → V3 Ultra-Executive
    │       │   └── NO → V1 VP/SVP Enterprise (default)
    │       └──
    └──
```

## Version Strengths

### V1 — VP/SVP Enterprise (2 pages)
- Full career narrative with quantified achievements
- Seven-team leadership emphasized
- $30M+ budget authority prominent
- Keytruda franchise ownership detailed
- HCP measurement frameworks highlighted
- ATS-optimized with keyword density
- **Use when:** Role JD emphasizes portfolio management, team scale, oncology depth

### V2 — Healthtech/SaaS Pivot (2 pages)
- Agency-to-tech translation table included
- Platform buyer perspective emphasized
- Workflow automation and process design highlighted
- Technology fluency section added
- Commercial engagement language (not "media planning")
- **Use when:** Company is a vendor/platform you previously bought from, or SaaS/healthtech

### V3 — Ultra-Executive (1 page)
- Scannable in 30 seconds
- Career trajectory format (not bullet-heavy)
- Board-ready tone
- Core authorities summary
- No fluff — pure signal
- **Use when:** Submitting through executive recruiter, high-volume screening, C-suite audience
