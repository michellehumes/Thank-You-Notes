## Page 2 — How to Use This Book

```
HOW TO USE THIS BOOK

This log is designed for families, home aides, and memory care supporters managing
daily care for an aging or ill loved one. Each daily spread captures one full day
of care—medications given, meals eaten, behaviors observed, and notes for the next
caregiver on duty.

Fill in the Care Team Reference Card on the next page before your first entry.
Complete each daily log at the end of every shift or day. When handing off to
another caregiver, circle any items needing follow-up. Keep this book in a
consistent, accessible location so every caregiver can find it quickly.
```
