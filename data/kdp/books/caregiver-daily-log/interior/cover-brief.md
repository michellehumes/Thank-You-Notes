# Cover Design Brief

**KDP trim size:** 6x9
**Page count:** 110
**Spine width formula:** (110 x 0.0025) + 0.06 = 0.335 inches

**Style:** Warm, compassionate — not clinical. Feels like a caring gift, not a medical form.
**Color palette:**
  - #8FAE8B — Soft sage green (primary background)
  - #F5F0E8 — Warm cream (text areas, panels)
  - #7A9CB8 — Dusty blue (accent, subtitle bar)
  - #C4A882 — Warm tan/caramel (borders, details)

**Typography:**
  - Title font: Montserrat Bold
  - Subtitle font: Inter Regular

**Cover imagery:** Two hands gently clasped — an older adult hand and a younger caregiver's hand — photographed from above on a soft cream or sage linen background. Warm, natural light. No medical equipment visible. Search Canva stock: "elderly hands held caregiver" or "senior hands holding". Alternative: a simple illustrated sage-colored background with a small line-art heart icon above the title.

**Required text:**
  - Title: Caregiver Daily Log
  - Subtitle: Elder Care, Memory Care & Daily Caregiving Journal
  - Author: Shelzy Perkins

**Back cover text:**
```
For every family managing care for an aging loved one —
this log brings clarity, continuity, and calm to every shift.

• Daily medication tracking with time, dose & caregiver initials
• Mood, behavior & orientation logs for memory care
• Meals, hydration & personal care checklists
• Vitals, pain scale & medical concern notes
• Handoff notes so every caregiver stays informed

110 pages | 6x9 inches | Paperback
```

**Canva steps:** Search '6x9 book cover' in Canva, select blank template, set canvas to 12.335 x 9 inches (front + spine + back), apply color palette above, place cover imagery on front panel, add title/subtitle/author text using specified fonts.
