## Klaviyo Setup Notes

- **Flow trigger:** List -- "Newsletter Subscribers" (or whatever list the site email capture feeds into)
- **Email 1:** Send immediately on subscribe
- **Email 2:** 2-day time delay after Email 1
- **Email 3:** 3-day time delay after Email 2 (Day 5 total)
- **Smart sending:** Enable to avoid over-emailing
- **UTM parameters:** Add utm_source=klaviyo, utm_medium=email, utm_campaign=welcome_series to all links
- **Coupon:** Create a static coupon code WELCOME15 for 15% off entire order, single use per customer
- **Expiration logic:** Email 3 mentions 48-hour expiry -- set the coupon to expire 7 days from subscribe date to give buffer
